## 🗺️ Sobre o Projeto

Esta landing page apresenta a **trilha DWF (Digital WorkForce)**, uma jornada de 5 etapas para a transformação digital de organizações com Microsoft 365 e IA. Cada serviço é um passo evolutivo, formando uma trilha conectada — do uso produtivo do M365 à criação de agentes de IA especializados via Copilot Studio.

- **Visual interativo:** Stepper de etapas, cards conectados, modo claro/escuro
- **Navegação fluida:** Clique nas etapas para navegar entre os serviços
- **Design responsivo:** Experiência otimizada para desktop e mobile

---

## 🪜 Trilha DWF — 5 Etapas

| Etapa | Serviço | Destaques |
|-------|---------|-----------|
| ①     | **Adoção Microsoft 365** | Teams, Outlook, SharePoint, OneDrive, base para IA |
| ②     | **Adoção Copilot Chat** | Trilha de IA, prompts, Agent Builder |
| ③     | **Copilot Premium** | Word, Excel, PowerPoint, Teams, Outlook, Studio |
| ④     | **Imersão Executiva** | Jornada consultiva, cocriação de cenários e agentes |
| ⑤     | **Agentes Especializados** | Desenvolvimento sob medida via Copilot Studio (em breve) |

## 🧩 Conteúdo da Página

- **Hero:** Título, subtítulo, estatísticas e partículas animadas
- **Stepper:** Visualização da trilha de 5 etapas (interativo)
- **Cards de Serviço:** 5 cards, cada um representando uma etapa da jornada, com badges de etapa e destaques
- **Diferenciais Lanlink:** Certificações, especializações e experiência Microsoft
- **Contato e Footer:** LGPD, escritórios, canais de contato
