## 📋 Sobre o Projeto

Esta é uma **landing page institucional** desenvolvida em HTML, CSS e JavaScript puro, criada para apresentar o catálogo de workshops do **Programa de Adoção de IA (Copilot Chat) com Responsabilidade e Segurança** da Lanlink.

O programa consiste em uma **trilha progressiva de 6 workshops temáticos** com duração de 1h30 cada, totalizando **10 horas de capacitação estruturada**, conduzidos por especialistas em adoção de Inteligência Artificial e Microsoft 365.

A proposta é voltada para organizações que já dispõem do **Microsoft 365 com Copilot Chat habilitado** e desejam maximizar o retorno sobre esse investimento por meio de uma adoção estruturada, orientada a resultados e alinhada às boas práticas de uso da IA no ambiente corporativo.

## 🚀 Como Usar

### ▶️ Localmente

```bash
# 1. Clone o repositório
git clone https://github.com/usuario/lanlink-copilot-adoption.git

# 2. Acesse a pasta
cd lanlink-copilot-adoption

# 3. Abra o arquivo no navegador
# Basta dar duplo clique em index.html
# ou usar uma extensão como Live Server no VS Code
```

### 🌐 Via GitHub Pages

1. Acesse o repositório no GitHub
2. Clique em **Settings** ⚙️
3. No menu lateral, clique em **Pages**
4. Em **Branch**, selecione `main` e a pasta `/ (root)`
5. Clique em **Save**
6. Aguarde alguns instantes e acesse:

```

```
